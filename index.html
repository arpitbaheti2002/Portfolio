Flight Deal Finder

This project uses Python to automate the process of finding the best round-trip flight deals for a given destination. By checking Google Sheets via Sheety API for the destination, the program then queries the Kiwi API to retrieve the cheapest flights available within the next few months. This project demonstrates the use of APIs and automation to simplify the flight search process for users.
